About

Lightweight multi-tab GUI terminal wrapper with clipboard image paste support. Works with any CLI tool — Claude Code, Codex, and more.

Usage

  • Click + or press Cmd+T to open a new session
  • Paste screenshots with Cmd+V — preview appears above terminal
  • Drag & drop files into window to attach
  • Press Enter to send with file path auto-attached
  • Cmd+W close tab, Ctrl+Tab switch tabs

Keyboard Shortcuts

  • Cmd+T / Ctrl+T — New tab
  • Cmd+W / Ctrl+W — Close tab
  • Ctrl+Tab — Next tab
  • Ctrl+Shift+Tab — Previous tab
  • Cmd+V — Paste image
  • Cmd+, — Settings
